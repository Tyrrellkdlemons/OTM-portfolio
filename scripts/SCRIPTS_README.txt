OTM-portfolio :: One-click scripts
====================================

Run scripts\00-menu.bat for the menu.

00-menu.bat              Main launcher (start here)
01-init-repo.bat         Initialize/repair git remote
02-commit-push.bat       Commit and push to main
03-deploy-netlify.bat    Deploy to Netlify production
04-dev-server.bat        Local server on http://localhost:8081
05-clean.bat             Remove local caches
06-fix-deps.bat          Reinstall Netlify CLI
07-open-browser.bat      Open local + live + GitHub + Netlify admin
08-status.bat            Git + Netlify status
10-first-time-setup.bat  Run once on a new machine

All scripts assume the repo root is the parent folder.
